Working with Files

Creating a New File

In the File tab, select New (Ctrl+N)

Opening a File

In the File tab, select Open (Ctrl+O), or drag and drop a file into the editor window

Saving a File

In the File tab, select Save (Ctrl+S) or Save As to save under a different name

Closing a File

In the File tab, select Exit (Ctrl+W) to close the open file

Text Editing (Edit)

Undo/Redo - undo the last action / redo an undone action

Cut - the selected text is cut and copied to the clipboard

Copy - the selected text is copied to the clipboard

Paste - pastes text from the clipboard into the editor window

Delete - deletes the selected text

Select All - selects all text in the editor window
